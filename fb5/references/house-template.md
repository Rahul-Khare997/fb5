# House standards (overlay template)

Copy this file to `references/house.md` and fill it in, or better, run the tailor
interview in [docs/TAILOR.md](../../docs/TAILOR.md) and let the model fill it from your
answers. When `house.md` exists, Phase 0 folds its rules into every rubric, and they
OUTRANK the craft defaults where they conflict, because your brand beats the generic bar.

Rules for filling it: same bar as the craft files. Every line a number, a named pattern,
or a binary condition; nothing the craft files already say; delete every section you have
no opinion on (an empty section is noise, not coverage). Keep the whole file under 60
lines; it is read on every run.
